## 项目提交流程

* 切换并且更新本地dev分支。
* 从dev分支，分一个自己的branch出来，可以起个名为 dev_hzheng。
* 在自己的分支下完成代码的修改。
* 将完成的代码修改合并到dev分支上。
* 所有人的代码开发完成并且合并到dev后，开始进行测试dev分支。
* dev测试通过，将dev合并到master上。
